## Getting Started

1. Read `README.md` for an overview
2. Follow `INSTALLATION.md` for step-by-step setup
3. Upload `SKILL.md` to Claude Code
4. Add activation instruction to your `CLAUDE.md`
5. Invoke the skill at the start of task-oriented sessions

## After Installation

Once installed, the skill will:
- Automatically create a `skill-observations/` directory in your workspace
- Begin logging observations silently during task sessions
- Surface insights at the end of each session
- Help you build and refine reusable skills
