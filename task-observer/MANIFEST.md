## Getting Started

1. Read `README.md` for an overview
2. Follow `INSTALLATION.md` — choose Personal or Work setup at the top
3. Upload `SKILL.md` to Claude Code or Claude.ai Web
4. Add activation instruction to your configuration
5. Invoke the skill at the start of task-oriented sessions

## After You're Up and Running

- **Making changes?** See `UPDATE-GUIDE.md` for how to sync SKILL.md updates to Claude Code
- **Work setup:** Changes sync automatically via git reference
- **Personal setup:** Re-upload updated SKILL.md or update inline in Project Instructions

## After Installation

Once installed, the skill will:
- Automatically create a `skill-observations/` directory in your workspace
- Begin logging observations silently during task sessions
- Surface insights at the end of each session
- Help you build and refine reusable skills
