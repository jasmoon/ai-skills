# Task Observer Skill

**Continuous Skill Discovery & Improvement**

*Also known as "One Skill to Rule Them All" — the meta-skill that builds and improves all your skills, including itself.*

Created by Eoghan Henn / [rebelytics.com](https://rebelytics.com)

## What This Skill Does

Task Observer formalizes the process of noticing skill improvement opportunities during real work. It captures patterns, user corrections, workflow insights, and methodology worth preserving as reusable skills — all without interrupting your workflow.

Use this skill during any multi-step task, agentic workflow, or substantive work session where you're using tools and producing deliverables.

## Installation

See [INSTALLATION.md](INSTALLATION.md) — covers both Personal (mobile + multiple desktops) and Work (local desktops only) setups, including syncing updates.

## Key Features

- **Silent observation** — captures insights without interrupting your work
- **Structured logging** — observations recorded with context and actionable suggestions
- **Multi-environment support** — works in Claude Code, Claude.ai, and Cowork
- **Built-in enforcement** — verifies that skills follow their own rules
- **Confidentiality safeguards** — keeps open-source and internal work properly separated
